# hero

## 技术栈

- 技术栈：react
- 构建工具：vite
- 可视化库：threejs、echarts、three.js
- ui 库：antd
- 样式：less
- 网络请求库：axios
- react hooks 库：ahooks
- 前端缓存库：localforage
- 工具库：lodash
- 语言 locale
- 状态库：jotai

## git commit 规范

- feat - 新功能 feature
- fix - 修复 bug
- docs - 文档注释
- style - 代码格式(不影响代码运行的变动)
- refactor - 重构、优化(既不增加新功能，也不是修复 bug)
- perf - 性能优化
- test - 增加测试
- chore - 构建过程或辅助工具的变动
- revert - 回退
- build - 打包
